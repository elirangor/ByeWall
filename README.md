# Page Rewind - Chrome Extension


üì¶ A browser extension that lets you quickly open archived versions of web pages using [Archive.today](https://archive.today) and [Wayback Machine](https://web.archive.org).

## ‚úÖ Features

- One-click access to archived pages
- Choose between Archive.today or Wayback Machine
- Track your last 5 archived pages
- Dark mode support
- Keyboard shortcuts: `Ctrl+Shift+Q` (open popup), `Ctrl+Shift+U` (instant open of latest snapshot)
- Option of choosing whether open the archived snapshot on a new tab or in the same one
